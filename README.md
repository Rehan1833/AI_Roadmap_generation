# AI Roadmap Generator

This project is an AI-based decision support system that generates
personalized career and learning roadmaps based on user skills.

## Features
- Skill analysis
- Career recommendation
- Step-by-step roadmap generation

## Technologies Used
- Python
- Pandas
- CSV Dataset

## How to Run
1. Install requirements
2. Run main.py
3. Enter skills
